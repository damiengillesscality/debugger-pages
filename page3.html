Step 1: Understanding the Code Flow

We have a chain of function calls that you'll step through. First, set a breakpoint in the code:

function processData(input) {
    debugger; // ‚Üê Automatic breakpoint here!
    const processed = transformValue(input);
    const result = calculateResult(processed);
    return finalizeResult(result);
}

function transformValue(value) {
    return value * 2;
}

function calculateResult(processed) {
    return processed + 10;
}

function finalizeResult(result) {
    console.log(`Final result: ${result}`);
    return result;
}

üéÆ Step Controls Overview

  • Step Over (F10) - Execute current line, don't go into functions
  • Step Into (F11) - Enter function calls to debug inside them
  • Step Out (Shift+F11) - Exit current function and return to caller
üìç Your Task:
  • Open the Developer Tools, it automatically pauses at the start of the function
  • Use the step controls until you get to the line where we compute the result (const result = value * 2;)
  • Click disable breakpoints and resume the code execution
